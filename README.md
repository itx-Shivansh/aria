# ARIA — Adaptive Reasoning & Intelligent Assistant

A personal AI desktop assistant built with Electron + React + Ollama.
Developed by **Shivansh Raj**.

## Features
- 💬 Natural language chat with local AI (100% free, no API key needed)
- 📱 WhatsApp automation — send messages by voice command
- 🧠 Persistent memory across sessions
- 💻 Code agent — write, debug, and run code
- 📧 Email drafting
- 📅 Calendar and scheduling

## Setup

### Requirements
- Node.js 18+
- Ollama (free, local AI): https://ollama.com/download

### Install
npm install

### Run
ollama serve         # Start AI engine (keep running)
npm run dev          # Start ARIA

### Pull AI Model (first time only)
ollama pull llama3

## Tech Stack
- Electron 31
- React 18
- Vite 5
- Ollama (local LLM)
- whatsapp-web.js
